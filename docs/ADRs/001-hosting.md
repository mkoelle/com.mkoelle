# Use of AWS for hosting

## Status

Accepted

## Context

My projects need an inexpensive location for hosting and management.
Additionally I want to showcase my skills and experience while exploring new technologies.

Options Considered:

- Amazon web services (AWS)
- Microsoft Azure (Azure)
- Google Cloud Platform (GCP)

### Reasoning to choose AWS over Azure and GCP

1. **Market Leader**: AWS is the market leader in cloud computing, with a wide range of services and a large customer base. This ensures stability, reliability, and continuous innovation.

2. **Global Infrastructure**: AWS has a global infrastructure with data centers in multiple regions, allowing you to deploy your applications closer to your target audience, resulting in lower latency and better performance.

3. **Service Offering**: AWS offers a comprehensive suite of services, including compute, storage, databases, networking, machine learning, and more. This allows you to build and scale your applications using a wide range of tools and technologies.

4. **Ecosystem and Integration**: AWS has a rich ecosystem of third-party integrations and partnerships, making it easier to integrate with popular tools and services. This enables seamless integration with existing systems and workflows.

5. **Cost-effective**: AWS offers flexible pricing options, including pay-as-you-go and reserved instances, allowing you to optimize costs based on your usage patterns. Additionally, AWS provides cost management tools to help you monitor and control your expenses.
