# Use of Liquid for template language

## Status

Accepted

## Context

Nunjucks vs Liquid

Liquid is now the default template language in Eleventy.

- [mortendk-from-nunjucks-to-liquid](https://morten.dk/post/from-nunjucks-to-liquid/)

- Liquid provides a more intuitive and user-friendly syntax compared to Nunjucks.
- Liquid has better documentation and a larger community, making it easier to find support and resources.
- Liquid offers more built-in filters and tags, providing greater flexibility in template customization.
- Liquid is the recommended template language for Eleventy, ensuring compatibility and future updates.
